Running out of Space Game Jam

Escape Small Slimes on a Randomly Generated Flooded Island

This was my semester final project for my game development class, I worked with a partner to create this game which uses perlin noise to create infinitely randomly generated terrain. Throughout the game, small slimes spawn nearby the player. Touching one of the slimes kills the player. As the game progresses, the island starts to sink, and the play runs out of space to escape the slimes.


Features and Usage

To play the game, go to this link. Click run game, wait for it to load, and click the start button. Then you can move around using WASD or the arrow keys to escape the slimes.

Godot Toy

An Environmental Ball Clicker

This project was a small game I made to practice lighting and other Godot features for class. I learned many new parts of Godot with this project, namely particles, scene switching, and lighting.


Features and Usage

To play the game, go to this link. Click run game, wait for it to load, then enter full screen. Click the start button then click any ball on screen to pop it.
