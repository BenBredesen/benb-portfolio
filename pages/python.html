Olympic Track and Field Trends

Graphical Evidence of Track and Field Innovation

As an introduction to data science, I made a series of python scripts which generate graphs using matplotlib and pandas. The produced graphs measure Olympic track and field times since 1896, that yield intriguing and consistent results, indicating a pretty clear and consistent improvement in race times over the last hundred years. The data represented in the graph could be useful to someone innovating the track and field sphere, as they could see which years had the largest improvements, and which innovations may have come around the time.


Features and Usage

To use the project, open this GitHub repository. Then, download either one of the files titled 'event.py', for example 400m.py, or download general.py which can display any event. Once downloaded, run the file in your terminal or IDE. The general.py file takes multiple inputs, allowing you to specify the event and number of events to graph.

Ball Breakout Game

A Simple, Fun Game

In Ball Breakout, the user must direct and buy bouncing balls to break blocks that spawn in waves across the board. The player gains money from each block broken and has to manage their funds effectively to balance buying and upgrading various types of balls. They also need to balance buying different types of balls as some balls work well against strong bricks and some balls work better against groups of bricks. Not only does the user buy balls, but each ball has various attributes that can be upgraded to improve anything from damage to accuracy.


Features and Usage

To play the game, go to this link. Follow the in-game tutorial to buy your first ball, then return to the main screen and watch it break the blocks. Return to the purchase page to buy more balls or visit the upgrade page to upgrade your existing balls. Ball Breakout is a single-player game that continues infinitely.

Shortcuts

To speed up the game, you can press Q to gain $100, or press W to spawn the next wave.

Markdown to HTML Converter

Convert Between Markdown Files and HTML Files

For this project, I worked with two other students in my class to create a python file that can convert basic markdown files into html code. Similar code to ours is used in most websites that can display markdown. For example, GitHub uses something similar to display readme.md files on their website.


Features and Usage

To use the converter, go to this GitHub repository. Download the markdown_converter.py file and run it in your terminal or IDE. It takes a list of markdown file names as an input, and will output a new html file for each inputted file.
