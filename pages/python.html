Olympic Track and Field Trends

Graphical Evidence of Track and Field Innovation

As an introduction to data science, I made a series of python scripts which generate graphs using matplotlib and pandas. The produced graphs measure Olympic track and field times since 1896, that yield intriguing and consistent results, indicating a pretty clear and consistent improvement in race times over the last hundred years. The data represented in the graph could be useful to someone innovating the track and field sphere, as they could see which years had the largest improvements, and which innovations may have come around the time.


Features and Usage

To use the project, open this GitHub repository. Then, download either one of the files titled 'event.py', for example 400m.py, or download general.py which can display any event. Once downloaded, run the file in your terminal or IDE. The general.py file takes multiple inputs, allowing you to specify the event and number of events to graph.
